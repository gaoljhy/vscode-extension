## Usage

打开命令行  `ctrl + p`
输入 `Set Clock` 

然后回车,选择需要设定的时间,即可

等待时间设定完成,即会提醒

### Command

打开命令条 输入 `Set Clock` ，即可翻译当前选中的内容
